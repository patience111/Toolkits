## Vim 
* 复制 
<br>yy/Y:复制当前光标所处在的行<br>
nyy/nY:复制当前光标所在行以及以下的n行

* 删除/剪贴<br>
dd:剪贴光标所在行
<br> ndd:剪贴当前光标所在以及以下的n行
* 粘贴<br>
p:在当前光标处下面粘贴内容
<br>P:在当前光标处上面粘贴内容

* 查找与替换 http://harttle.land/2016/08/08/vim-search-in-file.html
